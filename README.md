# Memory
- 使用cocos creator开发的小游戏
- [google play 地址：](https://play.google.com/store/apps/details?id=com.simple2l.memory)
- 玩法是需要回答前后显示的两张图片是否一样
- 可以使用钻石购买选择不同图片
- 道具有磁铁，生命，双倍积分和防雷
